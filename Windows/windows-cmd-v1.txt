Windows CMD

nützliche/ wichtige CMD Befehle:


Hilfe:
        $ help

Hilfe für einen Befehl:
        $ PLATZHALTER-BEFEHL /?

Clear:
        $ cls

Benutzer anzeigen:
        $ whoami

Abbrechen:
        Tastenkombinaton: ctrl + C

Farbe ändern:
        $ color /?
        $ color 0a

Herunterfahren/Neustart:
        $ shutdown
        $ reboot

Systeminformationen:
        $ systeminfo

GPU-Info:
        $ wmic path win32_VideoController get name

Netzwekeinstellungen:
        $ ipconfig
        $ ipconfig /all

Traceroute:
        $ tracert
        $ tracert www.google.de

Liste aller Geräte im Netzwerk:
        $ arp -a

Hintergrundprogramme, die Pakete senden überprüfen:
        $ netstat
        $ netstat -b -t

Dateien in einem Verzeichniss anzeigen:
        $ dir

neuen Ordner erstellen:
        $ mkdir ORDNER

Verzeichniss wechseln:
        $ cd /PFAD/zum/VERZEICHNIS
        $ cd ..
